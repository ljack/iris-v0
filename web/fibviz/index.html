How This App Is Built

        

          

            
1) Iris Source

            
fib_viz.iris emits EVT/METRIC lines while running Fibonacci algorithms.

          

          

            
2) Compiler → Wasm

            
The Iris compiler generates WebAssembly + a small host ABI.

          

          

            
3) Browser Host

            
app.js provides the host functions (print, strings, args) for the Wasm module.

          

          

            
4) Visualization

            
EVT/METRIC lines are parsed into charts and metrics on the page.

          

        

      

    


    

      

        Build the wasm with:
        bin/iris run-wasm examples/real/apps/fib_viz.iris --no-run --wasm-out web/fibviz/fib_viz.wasm
        (also writes fib_viz.wasm.b64)
